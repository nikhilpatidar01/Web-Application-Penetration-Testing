## ğŸ§¼ Cleaning Up Resources

* Always close resources after use using appropriate functions like:

  * `fclose()` for files
  * `mysqli_close()` for MySQL connections
  * `curl_close()` for cURL

```php
<?php

// ...use the file
fclose($handle);  // Proper resource cleanup
?>
```
