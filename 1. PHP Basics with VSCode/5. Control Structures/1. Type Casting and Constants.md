
# 📘 PHP Type Casting

> **Type Casting** means converting a variable from one data type to another.
> PHP supports:

* **Implicit Type Casting** (automatic)
* **Explicit Type Casting** (manual)

---

## 🔹 1. Implicit Type Casting (Automatic)

> PHP automatically converts types based on context (like math operations).

### 🔸 Example:

```php
<?php
$number = 10;      // Integer
$text = "5";       // String

$result = $number + $text; // PHP automatically casts "5" to integer

echo "Result: $result\n";             // Output: Result: 15
echo "Type: " . gettype($result);     // Output: integer
?>
```

### 🧾 Explanation:

* PHP saw `"5"` as numeric, so it converted it to integer automatically.
* This is **implicit casting**.

---

## 🔹 2. Explicit Type Casting (Manual)

> You can **manually** convert a value’s type using cast syntax: `(int)`, `(float)`, `(string)`, etc.

### 🔸 Example:

```php
<?php
$price = "99.99";                      // String

$price_float = (float)$price;         // To float
$price_int   = (int)$price;           // To integer
$price_str   = (string)$price_float;  // Back to string

echo "Float: $price_float\n";         // 99.99
echo "Integer: $price_int\n";         // 99
echo "String: $price_str\n";          // "99.99"
?>
```

---

## 🧪 Common Casts in PHP

| Cast Type | Syntax         | Description                      |
| --------- | -------------- | -------------------------------- |
| Integer   | `(int)$var`    | To integer                       |
| Float     | `(float)$var`  | To float                         |
| String    | `(string)$var` | To string                        |
| Boolean   | `(bool)$var`   | To boolean (true/false)          |
| Array     | `(array)$var`  | To array                         |
| Object    | `(object)$var` | To object                        |
| Null      | `unset($var)`  | Destroys variable (becomes null) |

---

## 💻 Practical Example – `casting.php`

```html
<!DOCTYPE html>
<html>
<head>
    <title>PHP Type Casting (Demo)</title>
</head>
<body>

<h2>🔁 PHP Type Casting Demo</h2>

<?php
// Implicit Casting
$a = "10";
$b = $a + 5;

echo "🔸 \$a = \"$a\" (type: " . gettype($a) . ")<br>";
echo "🔸 \$b = \$a + 5 = $b (type: " . gettype($b) . ")<br><br>";

// Manual Casting with settype()
$x = "25.99";
settype($x, "integer");

echo "🔸 \$x after settype() = $x (type: " . gettype($x) . ")<br><br>";

// Manual Casting using (int)
$y = (int)"50.75";
echo "🔸 \$y = (int)'50.75' = $y (type: " . gettype($y) . ")<br><br>";

// Boolean Casting
$z = (bool) 0;
$w = (bool) 1;

echo "🔸 (bool)0 = " . ($z ? "true" : "false") . " (type: " . gettype($z) . ")<br>";
echo "🔸 (bool)1 = " . ($w ? "true" : "false") . " (type: " . gettype($w) . ")<br>";
?>

</body>
</html>
```

---

## ✅ Summary

* **Implicit casting**: Done by PHP automatically.
* **Explicit casting**: Done manually using casting syntax or `settype()`.
* Helps in better type handling and avoids errors.

# 📘 PHP Constants

## 🔹 What is a Constant?

A **constant** is a value that **cannot be changed** once it's defined.
Used for fixed values like site settings, version numbers, etc.

---

## 🔑 Key Features

* Declared using `define()` or `const`
* **No** `$` sign before the name
* **Global scope**
* Cannot be redefined or changed
* Naming is usually **UPPERCASE**

---

## 🔧 Syntax

### Using `define()` (global)

```php
define("SITE_NAME", "MyWebsite");
```

### Using `const` (PHP 5.3+)

```php
const AUTHOR = "Nikhil";
```

> 🔸 `define()` is used outside classes
> 🔸 `const` can be used inside classes or globally (PHP 5.3+)

---

## 💡 Example – PHP Constants

```html
<!DOCTYPE html>
<html>
<head>
    <title>Constants in PHP</title>
</head>
<body>

<h2>🔐 PHP Constants Example</h2>

<?php
// Define using define()
define("SITE_NAME", "MyWebsite");
define("MAX_USERS", 500);

echo "SITE_NAME: " . SITE_NAME . "<br>";
echo "MAX_USERS: " . MAX_USERS . "<br>";

// Define using const
const AUTHOR = "Nikhil Patidar";
const VERSION = "2.0";

echo "AUTHOR: " . AUTHOR . "<br>";
echo "VERSION: " . VERSION . "<br>";

// Types of constants
echo "Type of SITE_NAME: " . gettype(SITE_NAME) . "<br>";
echo "Type of MAX_USERS: " . gettype(MAX_USERS) . "<br>";
?>

</body>
</html>
```

---

## 📌 Quick Comparison

| Feature         | `define()`   | `const`            |
| --------------- | ------------ | ------------------ |
| PHP Version     | All versions | PHP 5.3+           |
| Scope           | Global       | Global / Class     |
| Re-assignable   | ❌ No         | ❌ No               |
| Syntax Type     | Function     | Language Construct |
| Usable in Class | ❌ No         | ✅ Yes              |
