## üîÑ CRUD Operations

CRUD refers to the four basic operations on a database:

* **C**reate ‚Äì `INSERT` new records
* **R**ead ‚Äì `SELECT` data
* **U**pdate ‚Äì `UPDATE` records
* **D**elete ‚Äì `DELETE` records

---

## üß© PHP Database Interaction in 5 Steps (Using MySQLi)

### ‚úÖ Step 1: Create a Database Connection

```php
$connection = mysqli_connect("localhost", "root", "password", "my_database");

// Check connection
if (!$connection) {
    die("Database connection failed: " . mysqli_connect_error());
}
```

### ‚úÖ Step 2: Perform a Database Query

```php
$query = "SELECT * FROM users";
$result = mysqli_query($connection, $query);
```

### ‚úÖ Step 3: Use Returned Data (If Any)

```php
while ($row = mysqli_fetch_assoc($result)) {
    echo "User: " . $row["username"] . "<br />";
}
```

> You can also use:
> `mysqli_fetch_row()` ‚Äì returns indexed array
> `mysqli_fetch_array()` ‚Äì returns both associative & indexed arrays

### ‚úÖ Step 4: Release Result Memory

```php
mysqli_free_result($result);
```

### ‚úÖ Step 5: Close the Database Connection

```php
mysqli_close($connection);
```
