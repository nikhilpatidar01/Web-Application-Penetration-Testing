
# üëÄ What is `View-Records-with-PHP`?

`View-Records-with-PHP` is a **PHP script** that **fetches data from a MySQL database** and displays it in an HTML table on a web page.

---

## ‚úÖ `view-records.php`

```sql
<?php
// DB Connection
$dbhost = "localhost";
$dbuser = "root";
$dbpass = "root";
$dbname = "nikhil_db";
$connection = mysqli_connect($dbhost, $dbuser, $dbpass, $dbname);

if (mysqli_connect_error()) {
    die("Database Connection Failed: " . mysqli_connect_error());
}
?>

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>View Records with PHP</title>
    <style>
        body { font-family: Arial, sans-serif; padding: 20px; }
        table {
            border-collapse: collapse;
            width: 100%;
            margin-top: 20px;
        }
        th, td {
            border: 1px solid #444;
            padding: 8px;
            text-align: left;
        }
        th {
            background-color: #eee;
        }
    </style>
</head>
<body>

<h1>View Records with PHP</h1>

<table>
<thead>
<tr>
    <th>ID</th>
    <th>NAME</th>
    <th>AGE</th>
    <th>GENDER</th>
    <th>EMAIL</th>
    <th>COURSE NAME</th>
    <th>CITY NAME</th>
    <th>STUDENT NAME</th>
    <th>STUDENT AGE</th>
    <th>PERCENTAGE</th>
</tr>
</thead>
<tbody>

<?php
// Run the corrected SQL query
$query = "
    SELECT
        u.id AS user_id,
        u.name AS user_name,
        u.age AS user_age,
        u.gender AS user_gender,
        u.email AS user_email,
        co.course_name,
        ci.city_name,
        s.name AS student_name,
        s.age AS student_age,
        s.percentage
    FROM users u
    INNER JOIN course co ON u.course_id = co.course_id
    INNER JOIN city ci ON u.city_id = ci.city_id
    LEFT JOIN student s ON s.email = u.email
";

$result = mysqli_query($connection, $query);

// Check for result
if (!$result) {
    die("Database Query Failed: " . mysqli_error($connection));
}

// Output each row
while ($row = mysqli_fetch_assoc($result)) {
    echo "<tr>";
    echo "<td>{$row['user_id']}</td>";
    echo "<td>{$row['user_name']}</td>";
    echo "<td>{$row['user_age']}</td>";
    echo "<td>{$row['user_gender']}</td>";
    echo "<td>{$row['user_email']}</td>";
    echo "<td>{$row['course_name']}</td>";
    echo "<td>{$row['city_name']}</td>";
    echo "<td>" . ($row['student_name'] ?? '-') . "</td>";
    echo "<td>" . ($row['student_age'] ?? '-') . "</td>";
    echo "<td>" . ($row['percentage'] ?? '-') . "</td>";
    echo "</tr>";
}
?>

</tbody>
</table>

<?php
mysqli_free_result($result);
mysqli_close($connection);
?>

</body>
</html>

```
